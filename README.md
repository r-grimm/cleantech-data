# cleantech-data

A repository of cleantech and decarbonisation data.

## Structure

- `/catalog/datasets.yaml` - Dataset catalog with metadata
- `/data/*.csv` - Data files

## Adding New Datasets

1. Add entry to `catalog/datasets.yaml`
2. Run ingest script to fetch data
3. Commit and push data file

## Catalog Format

See `catalog/datasets.yaml` for dataset entry structure.

## Data Sources

- **SMARD** (Bundesnetzagentur): https://www.smard.de
- **Energy-Charts.info** (Fraunhofer ISE): https://energy-charts.info
