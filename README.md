# cleantech-data

A data repository of cleantech and decarbonization datasets, focused on German renewable energy, wind capacity, solar, battery storage, and related metrics. Serves as the single source of truth for the [Cleantech Chart Builder](https://github.com/r-grimm/chart-engine).

## Structure

```
data/
├── catalog.json                    # Auto-generated index (24+ datasets)
├── power_mix.csv                   # German electricity generation (SMARD)
├── 2025-11/windpower/              # Wind capacity data
├── 2025-12/                        # Battery, wind, solar, steel datasets
│   ├── batteries/
│   ├── windpower/
│   ├── solar/
│   └── steel/
└── 2026-01/                        # Recent data
    ├── charts/                     # OCR-extracted chart data
    └── imported/                   # External source imports
catalog/
└── datasets.yaml                   # Dataset metadata definitions
scripts/
└── rebuild_catalog.py              # Regenerate catalog.json
```

## Dataset Format

Each dataset consists of:
- **CSV file** at `data/YYYY-MM/{thematic_name}/filename.csv`
- **Metadata sidecar** at `data/YYYY-MM/{thematic_name}/filename.meta.json`

Metadata includes title, description, tags, source URL, import type (csv/ocr), row count, and column definitions.

## Adding New Datasets

1. Add entry to `catalog/datasets.yaml`
2. Place CSV + `.meta.json` in the appropriate `data/YYYY-MM/{theme}/` directory
3. Commit and push -- catalog rebuilds automatically via CI

## Catalog

The catalog (`data/catalog.json`) is auto-generated by `scripts/rebuild_catalog.py`. It indexes all datasets with:

- Path, title, description, tags
- Source information (name, URL, import type)
- Freshness tracking fields
- Thematic grouping and date organization

### Rebuild manually

```bash
python scripts/rebuild_catalog.py
```

No external dependencies required -- uses Python stdlib only.

## CI/CD

GitHub Actions (`.github/workflows/update-catalog.yml`) automatically rebuilds `catalog.json` when CSV or metadata files are pushed to the `data/` directory.

## Datasets

**Energy Generation:** German Power Mix (SMARD, daily updates)

**Wind:** Onshore Wind Development, Offshore Wind Capacity 2009-2024

**Solar & Batteries:** Solar Capacity/Zubau, Large-Scale Battery Storage, Grid Connection Requests, Utility-Scale BESS

**Infrastructure:** Land Use for Renewables, Steel Industry GDP Share

**Funding:** Cleantech Financing Germany 2020-2024

**OCR-Extracted Charts:** Copper Supply, Road Fuel Burn, Solar Power Boom, India Electrotech, Asia Electrification

## Data Sources

- **SMARD** (Bundesnetzagentur): https://www.smard.de
- **Energy-Charts.info** (Fraunhofer ISE): https://energy-charts.info
- **Fachagentur Wind und Solar**: Wind/Solar monitoring
- **Battery-Charts.de**: Battery storage tracking
- **Our World in Data**: Global historical data
- **Wikipedia/IRENA**: Renewable energy capacity
